🐛 Common Issues & Solutions

Issue: Dashboard tidak bisa diakses
Solution: Pastikan sudah login terlebih dahulu

Issue: Data tidak sync
Solution: Refresh browser atau clear localStorage

Issue: Modal tidak muncul
Solution: Check console errors dan pastikan script loaded

Issue: Export tidak berfungsi
Solution: Check browser support untuk download file
